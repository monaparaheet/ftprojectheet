## Getting Started

#descrition 

This Flutter application integrates a public REST API (https://dummyjson.com/products/category/groceries) to fetch and display grocery products dynamically. The app parses JSON data and presents products in a responsive GridView layout.

Each product card displays the product image, title, and price. The app uses FutureBuilder for asynchronous data handling and adapts the grid layout based on screen size for better responsiveness.

The project demonstrates API integration, JSON parsing, responsive UI design, and dynamic data rendering in Flutter.
